# Rename
##### by Emily Jimenez

### Description

Simple node JS template for building projects

## Setup/Installation Requirements
* Clone to desktop
* Open with text Editor
* Open terminal
* type npm install and wait for it to complete
* type bower install and wait for it to complete
* type gulp serve

## Technologies Used

* HTML
* CSS
* JavaScript
  * JQuery
  * Node.js
    * Gulp
    * Bower
    * Babel
* Google Chrome
